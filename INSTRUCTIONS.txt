📋 STEP 1: START THE DASHBOARD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Option A (Easiest):
   → Double-click "START_DASHBOARD.bat"
   → Dashboard opens automatically in your browser

   Option B (If Python is installed):
   → Open Command Prompt in this folder
   → Type: python -m http.server 8000
   → Open browser: http://localhost:8000/dashboard.html

   Option C (If Node.js is installed):
   → Open Command Prompt in this folder
   → Type: npx http-server -p 8000
   → Open browser: http://localhost:8000/dashboard.html


📊 STEP 2: UPDATE YOUR DATA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. Open "data/projects_data.csv" in Excel or Notepad
   2. Edit the project information
   3. Save the file
   4. Refresh your browser (Press F5)
   5. Dashboard updates automatically!

📝 DATA FORMAT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Your CSV file should have these columns:
   
   • Project_ID          - Unique ID (e.g., P001)
   • Project_Name        - Name of the project
   • Region             - North America, Europe, Asia Pacific, Latin America
   • Market             - Enterprise, Mid-Market, SMB
   • Industry           - Banking, Healthcare, Retail, etc.
   • AI_Category        - Agentic AI, AI Ops, Business Transformation, ICA
   • Agentic_AI         - Yes or No
   • AI_Ops             - Yes or No
   • Business_Transformation - Yes or No
   • ICA                - Yes or No
   • Budget             - Number (e.g., 500000)
   • Status             - Active, Completed, etc.
   • Start_Date         - YYYY-MM-DD format

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Problem: Dashboard shows "Loading data..."
   Solution: Make sure you're running a local server (use START_DASHBOARD.bat)

   Problem: Can't start the server
   Solution: Try Option B or C above, or install Python from python.org

   Problem: Changes not showing
   Solution: Press F5 to refresh the browser after saving CSV

   Problem: Charts not displaying
   Solution: Check your internet connection (needed for Chart.js library)
